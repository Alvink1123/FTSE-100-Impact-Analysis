# FTSE-100-Impact-Analysis

This project investigates how macroeconomic variables influence the return of the FTSE 100 index. Through Python, I collect, visualise, and analyse financial and economical data, to explore the relationship between stock market performance and changes in the broader economy.

Key macroeconomic variables including interest rates, inflation, unemployment, exchange rates, and other relevant factors are the focus of this analysis. Furthermore, statistical techniques namely descriptive, inferential, and explorative analysis methods are used to determine the direction of the relationship. 

This project was developed to enhance my skills and knowledge in financial and economics field, as well as the ability to utilise Python in data analysis, while demonstrating a clear data workflow suitable in financial and economical portfolio.
